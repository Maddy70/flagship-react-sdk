## Try it locally

### Prerequisites

-   **Node.js**: version 6.0.0 or later...

-   **Npm**: version 5.2.0 or later...

## Getting Started

-   **Install** the node module:

```
examples/react-ecommerce-demo$ npm install
```

-   **Start** the project:

```
examples/react-ecommerce-demo$ npm run start
```
